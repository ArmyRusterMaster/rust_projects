# Auth Service Docs

Компактная документация проекта после консолидации.

## Файлы

- `current-state.md` - актуальное состояние реализации и оставшиеся ограничения.
- `security-architecture-summary.md` - сжатая сводка по архитектуре, security baseline и roadmap.
- `protocols-summary.md` - сжатая сводка OAuth2/OIDC/JWT/Sessions/SAML/WebAuthn.
- `static-linking.md` - статическая линковка через `musl`.
- `production-roadmap.md` - поэтапный план доведения до production-ready платформы.

## Как читать

1. Начать с `current-state.md`.
2. Для архитектурных решений смотреть `security-architecture-summary.md`.
3. Для протокольных требований смотреть `protocols-summary.md`.
4. Для сборки бинарника смотреть `static-linking.md`.
